A class that represents a Pokémon from a Game. Extends the Species class. nickName is the name of the Pokémon. If the Pokémon has no nickname, this will be the same as the species name. speciesName is the name of the species. id is the id of the Pokémon. baseStats is the base stats of the species. ivStats is the IVs of the Pokémon. evStats is the EVs of the Pokémon.
